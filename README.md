# waveformCGI
a Python-Skript intended to work with CGI on Apache to generate waveform data compatible with (wavesurfer.js)[https://github.com/katspaugh/wavesurfer.js]

## Installation
1. Copy the script in your cgi-bin folder
2. Adjust the path variables _basepath_ and _filepath_ with the absolute path to the script folder and the path to the audio file respectively
3. install all necessary libraries in the folder _site-packages_
